# @aiolosjs/components

基于antd二次开发的组件库.

[![NPM version][image-1]][1] [![NPM downloads][image-2]][2]


## ✨ 特性

* 基于antd二次开发的表单组件
* 使用 TypeScript 构建，提供完整的类型定义文件。

## 📦 安装

```
npm i @aiolosjs/components --save
```

## 🔨 使用

```
import { ASelect } from '@aiolosjs/components';
```
